## 博客项目

- 使用koa2框架搭建
- 数据库：mySql，redis
- 反向代理：nginx

## 和express的区别

- express中间件是异步回调，koa2原生支持async/await
- 新开发框架和系统，都开始基于koa2
- express没有过时，但是koa2是未来的趋势
- koa2使用起来比express简约

## 脚手架创建项目

- npm i koa-generator -g
- koa2 'name'

## 主要功能

- 处理http接口（登录，查看博客，新建博客，更新博客，删除博库）
- 链接数据库
- 实现登录
- 安全
- 日志
